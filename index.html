Welcome to Raffle Simulator

Raffle simulator was a website created to simulate a random winner where participants have differnt chances of winning. This was originally created as a way for me to help out my school but is now being turned into a full fledged site.

Currently we only accept CSV files that are comma delimeted and text input but more file possilities are on their way!

Any developers check out the Github because this is written far from perfectly.
